# Tuya Dashboard

A web dashboard for inspecting and controlling your Tuya smart devices, running directly on your Home Assistant instance.

## Installation

1. Go to **Settings** > **Add-ons** > **Add-on Store**
2. Click the three dots menu (top right) > **Repositories**
3. Add: `https://github.com/GMetaxakis/tuya-dashboard`
4. Find **Tuya Dashboard** in the store and click **Install**
5. Start the add-on and click **Open Web UI**

## Usage

Log in with your Tuya IoT Platform credentials:

- **Access ID** and **Access Secret** from [iot.tuya.com](https://iot.tuya.com)
- Select your **Region** (EU, US, CN, or IN)

Your credentials are encrypted and stored in your browser — they are never saved on the server.

## Prerequisites

You need a Tuya IoT Platform Cloud Project with these API subscriptions:

- IoT Core
- Authorization Token Management
- Smart Home Basic Service
